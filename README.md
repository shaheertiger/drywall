# Primex Dry Wall - Vite + Vercel

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Upload this project to GitHub.
2. Import the repo into Vercel.
3. Vercel should detect Vite automatically.
4. Build command: `npm run build`
5. Output directory: `dist`

## Project structure

- `index.html`
- `src/main.jsx`
- `src/App.jsx`
- `src/styles.css`
- `vercel.json`
